# filmPediea

[Try it yourself!](https://kyzinatra.github.io/movieSercher/)

To start the program in dev mode:

```sh
  git clone https://github.com/kyzinatra/movieSercher.git
  cd filmPediea
  npm install
  npm run dev
```

To build the program:

```sh
  git clone https://github.com/kyzinatra/movieSercher.git
  cd filmPediea
  npm install
  npm run build
```

it could be useful for errors.
